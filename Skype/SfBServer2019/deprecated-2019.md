# Skype for Business Server 2019 deprecations

[!INCLUDE [disclaimer](disclaimer.md)]

The following features are found in Skype for Business Server 2015 but have been removed from Skype for Business Server 2019: 

- Persistent chat
- XMPP Gateways
- SQL Mirroring
- In-place upgrades from previous versions
- Support for TLS 1.0 and 1.1
