# Toll free dialing restrictions in Panama

|**Details**|**Description**|
|:-----|:-----|
|Is toll-free calling available?  <br/> |Yes  <br/> |
|Dialing format(s)  <br/> | 00-800-XXX-XXXX <br/> |
|Country/region ISO code  <br/> |PA  <br/> |
|Island/territory covered  <br/> |Not applicable  <br/> |
|Island/territory not covered  <br/> |Not applicable  <br/> |
|Wired/fixed line availability  <br/> |From C&amp;W only.  <br/> |
|Mobile network availability  <br/> |No  <br/> |
|Comments, restrictions, and access issues  <br/> |The customer must dial 011 (the international long-distance Panama code) before the toll-free number.  <br/> |
