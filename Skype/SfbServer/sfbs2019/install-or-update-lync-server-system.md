# Install or Update Lync Server System
[]
To begin the installation, or to update an existing deployment, of a Lync Server 2013 server, click **Install or Update Lync Server System**. The steps in this section will install components specific to a given server role, based on the Topology Builder document. Before you begin this series of steps, you must define the deployment in Topology Builder. You must be logged into the computer that you are deploying as the local administrator and as a member of the Domain Admins group in the domain where the server is located. The computer must be joined to the domain prior to completing this step.
  
If you are updating a currently deployed server, you must make the changes that you intend to make in Topology Builder.
