# Select a Region Link
[]
Use the **Region Link** dialog box to find region links available in your Lync Server 2010 environment. To use an existing region link, click a region link in the list and then click **OK**.
  
> [!NOTE]
> To retrieve all available region links, do not type anything in the search box and then click the search icon. 
  
For details about the different procedures that you can perform by using the Lync Server 2013 Control Panel, see [Operations in Lync Server 2013](operations.md).
