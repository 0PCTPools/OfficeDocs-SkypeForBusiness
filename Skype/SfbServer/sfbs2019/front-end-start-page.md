# Front End Start Page
[]
Use this wizard to define a new Front End pool for a site. Each site must have at least one Enterprise Edition pool or Standard Edition server. For details about the requirements for using the wizard to create a new Front End pool, see [Define and configure a Front End pool or Standard Edition server in Lync Server 2013](define-and-configure-a-front-end-pool-or-standard-edition-server.md) in the Deployment documentation.
