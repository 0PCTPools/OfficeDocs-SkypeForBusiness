# Legacy Merge Add Expanded Edge
[]
Lync Server 2013 does not support merging a legacy Office Communications Server 2007 R2 expanded Edge Server configuration. The expanded Edge Server must first be converted to a consolidated Edge Server configuration prior to merging the Edge Server information.
