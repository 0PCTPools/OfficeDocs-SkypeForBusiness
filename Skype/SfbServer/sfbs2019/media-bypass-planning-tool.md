# Media Bypass (Planning Tool)
[]
Media bypass refers to removing the Mediation Server from the media path whenever possible for calls whose signaling traverses the Mediation Server. For more details, see [Planning for media bypass in Lync Server 2013](planning-for-media-bypass.md).
