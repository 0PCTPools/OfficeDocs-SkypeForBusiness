# Add Archiving Server
[]
To deploy an Archiving Server into your topology design, you need to specify the fully qualified domain name (FQDN) of the server on which you want to deploy Archiving. Prior to publishing a topology that includes Archiving and installing Lync Server 2010 on the server, you should have completed all prerequisites for deploying Archiving.
