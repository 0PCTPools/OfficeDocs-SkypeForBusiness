# Deploy the Online Meeting Add-in for Lync 2013
[]
Deploy the Online Meeting Add-in for Lync 2013 so that users can schedule conferences that support dial-in conferencing.
  
The Online Meeting Add-in for Lync 2013 is installed automatically when you install Lync 2013. For details about customizing Online Meeting Add-in for Lync 2013 for Lync, see [Configuring the meeting invitation in Lync Server 2013](configuring-the-meeting-invitation.md).
