# Create Persistent Chat Servers Scope Node
[]
You right click on **Persistent Chat Pool** in Topology Builder to perform the following: 
  
- **New Persistent Chat Pool**: Starts the **Define New Persistent Chat Pool** wizard used to define a new Persistent Chat Server or pool. 
    
In addition to the New Persistent Chat Pool option, you can perform the following general tasks:
  
![Persistent Chat Servers Scope Node](media/Persistent_Chats_Scope_Node.jpg)
  
- **Topology**. You can use this option to define a new topology, open an existing topology, download the current topology, save a copy of the topology, publish the topology, install a database, merge components from previous versions of Office Communications Server, or remove the deployment.
    
- **Help**. You can use this option to access Help for the active node.
