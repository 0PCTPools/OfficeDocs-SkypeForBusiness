# Trunks Scope Node
[]
Right-click **Trunks** in Topology Builder to perform the following: 
  
- **New Trunk**: Starts the **Define New Trunk** wizard used to define a new SIP trunk. 
    
In addition to the New Trunk option, you can perform the following general tasks:
  
![Trunks Scope Node](media/Trunks_Scope_Node.jpg)
  
- **Topology**. You can use this option to define a new topology, open an existing topology, download the current topology, save a copy of the topology, publish the topology, install a database, merge components from previous versions of Office Communications Server, or remove the deployment.
    
- **Help**. You can use this option to access Help for the active node.
