# Apply Office Communications Server 2007 R2 updates [OCS 2007 R2 to W15]
[]
Before you migrate to Lync Server 2013, updates must be applied to your Office Communications Server 2007 R2 environment. For the most up-to-date information about Office Communications Server 2007 R2, see **Updates for** **Communications Server 2007 R2** at [https://go.microsoft.com/fwlink/p/?linkid=3052&amp;kbid=968802](https://go.microsoft.com/fwlink/p/?linkid=3052&amp;kbid=968802).
  
To install updates for Office Communications Server 2007 R2, we recommend you follow the **Method 1 Cumulative Server Update Installer** procedure described in Microsoft Knowledge Base article 968802, "Updates for Communications Server 2007 R2," at [https://go.microsoft.com/fwlink/p/?linkid=3052&amp;kbid=968802](https://go.microsoft.com/fwlink/p/?linkid=3052&amp;kbid=968802).
