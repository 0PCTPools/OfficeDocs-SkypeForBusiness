# Single Director in Lync Server 2013
[]
The Director can be deployed in either a single Director configuration or as a Director pool. This section defines a topology and configuration for a single Director. If you are planning on deploying a pool of Directors for the purposes of handling higher capacity and for high availability, see the topic [Scaled Director pool in Lync Server 2013](scaled-director-pool.md) for planning considerations for that topology.
