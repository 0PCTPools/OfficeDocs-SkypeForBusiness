# Edge Next Hop Settings Expander
[]
To set or modify the **Next hop pool** that the Edge Servers will use to communicate to the internal Lync Server 2013 infrastructure, select a Director, Director pool, Front End Server, or Front End Server pool, from the list. Only Directors or Front End Servers that have been configured in Topology Builder appear in the list. 
  
For details about defining and configuring the settings for Edge services, see [Define your edge topology in Lync Server 2013](define-your-edge-topology.md).
