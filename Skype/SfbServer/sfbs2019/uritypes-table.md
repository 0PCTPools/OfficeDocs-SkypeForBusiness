# UriTypes table in Lync Server 2013
[]
The UriTypes Table contains the different URI (Uniform resource identifier) types monitored in Microsoft Lync Server 2013.
  
|**Column**|**Data Type**|**Key/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**UriTypeId** <br/> |tinyint  <br/> |Primary  <br/> |Unique identifier assigned to a URI type.  <br/> |
|**UriType** <br/> |nvarchar(256)  <br/> || Descriptions of the different URI types. Allowed values are:  <br/>  1 - Phone Uri  <br/>  0 - User Uri  <br/> |
